# Google Docs Chrome Extension Experiment

Although not yet fully perfected, this is one of my personal experiences with DOM manipulation via Javascript/CSS/HTML. The Chrome extension provides a toggle for styling the Google Docs homepage in a dark theme. This simple application allows users to have a dark mode appearance on their Google Docs pages. To activate, the user must refer to their Chrome extensions and toggle the dark theme via the popup.  

Instructions:
-Download the repository
-Extract everything from the zip file
-Go to extensions and click on "manange extensions"
-Make sure "developer mode" is on
-Press on "load unpacked" and upload the file in
